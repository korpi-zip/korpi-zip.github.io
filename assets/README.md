# Recursos de Imágenes - Assets

Esta carpeta contiene las imágenes y recursos visuales del portafolio.

## Imágenes Recomendadas

### 1. Foto de Perfil (Opcional pero recomendado)
- **Nombre**: `profile.jpg` o `profile.png`
- **Ubicación**: `assets/images/profile.jpg`
- **Tamaño recomendado**: 400x400px o 600x600px
- **Formato**: JPG o PNG
- **Uso**: Se mostrará en la sección Hero si está disponible
- **Peso máximo**: 200KB (optimizar para web)

### 2. Favicon (Recomendado)
- **Nombre**: `favicon.ico`
- **Ubicación**: `assets/images/favicon.ico` o raíz del proyecto
- **Tamaño**: 32x32px o 16x16px
- **Formato**: ICO, PNG o SVG
- **Uso**: Icono que aparece en la pestaña del navegador

### 3. Open Graph Image (Opcional)
- **Nombre**: `og-image.jpg` o `og-image.png`
- **Ubicación**: `assets/images/og-image.jpg`
- **Tamaño recomendado**: 1200x630px
- **Formato**: JPG o PNG
- **Uso**: Imagen que se muestra al compartir el portafolio en redes sociales
- **Peso máximo**: 300KB

### 4. Logo Personalizado (Opcional)
- **Nombre**: `logo.svg` o `logo.png`
- **Ubicación**: `assets/images/logo.svg`
- **Tamaño**: Variable (SVG recomendado para escalabilidad)
- **Formato**: SVG o PNG
- **Uso**: Logo personalizado para el navbar (actualmente usa texto)

## Estructura de Carpetas Recomendada

```
assets/
├── images/
│   ├── profile.jpg          # Foto de perfil
│   ├── favicon.ico          # Favicon
│   ├── og-image.jpg         # Open Graph image
│   └── logo.svg             # Logo personalizado
└── README.md                # Este archivo
```

## Notas

- Todas las imágenes son **opcionales**. El portafolio funcionará perfectamente sin ellas.
- Si no agregas imágenes, el diseño usará iconos de Font Awesome y elementos CSS.
- Optimiza las imágenes antes de subirlas para mejorar el rendimiento.
- Usa herramientas como [TinyPNG](https://tinypng.com/) o [Squoosh](https://squoosh.app/) para comprimir imágenes.
